# VolumeCrisis

A SwiftUI iOS app to control in-app audio volume, set volume ceilings, manage user profiles and presets.

## Getting Started

1. Clone this repo.
2. Add a file named `test.mp3` to your Xcode project (for sound testing).
3. Open the project in Xcode and run on your device.

## Features

- In-app audio playback and volume control
- Volume ceiling
- User profiles and volume presets

> Note: The app cannot control the system or other app volumes due to iOS limitations.
